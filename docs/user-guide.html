这是什么01 概览

两个人互相监督的每日打卡工具。各自安排今天要做的事，做完了打个卡，对方立刻能看到；谁超时没做，对方那边会显示拖延了多久。不用群里报备，翻开就知道对方进度。

模板设置07 日常安排

点顶部记事本图标进模板页。工作日、周末各一份，新的一天打开App时会照模板自动生成当天安排。改模板只影响以后的新日子，今天已经生成的安排要在主页面里单独改。

常见问题08 FAQ

忘记密码怎么办

找建账号的人，账号和密码都是在Supabase后台手动建的，App本身没有找回密码的功能。

为什么看不到对方今天的安排

大概率对方今天还没打开过App。当天的安排是打开App那一刻才生成的，对方没登录过，那天就还没有记录。

拖延是怎么算的

从时段结束时间开始算，还在时段进行中不算拖延。比如8点到9点的安排，8点半还没做完不算数，9点还没打卡才开始计时。

对方能看到我的密码或者改我的安排吗

不能。对方只能看你打卡的结果，改不了你的时段，密码只有你自己知道，双方都是各自登录各自的账号。

手机和电脑看到的不一样吗

功能一样，手机上是切换标签页看我的/TA的，电脑屏幕够宽会直接左右两栏都显示，不用切换。

怎么退出登录

点顶部右上角的头像圆圈，会弹出一个菜单，里面有"退出登录"。

页面上出现"连接断了，正在重连"是什么意思

网络临时断开导致实时同步暂时收不到对方的更新，App在后台自动重连，一般几秒内恢复，不用手动刷新，恢复后这条提示会自己消失。
